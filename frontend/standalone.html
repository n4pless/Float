📝 Code Structure

/sdk/drift-client-wrapper.ts - TypeScript SDK wrapper with methods:

  • openLongPosition() - Enter long trades
  • openShortPosition() - Enter short trades
  • closePosition() - Exit positions
  • getAccountState() - Fetch account data
  • getPositions() - List open positions

/frontend - React + Vite application:

  • App.tsx - Main component with wallet provider
  • components/ - UI components (Trading, Positions, Portfolio)
  • hooks/ - Custom React hooks for Drift integration
  • config.ts - Configuration for network and trading parameters

⚡ Note: Windows npm Issues

Due to Windows permission issues with deep npm dependencies, try these solutions:

  • npm install --legacy-peer-deps
  • npm install --force
  • Run PowerShell as Administrator
  • Use WSL2 Ubuntu terminal instead
